    Wireframes (The First Steps)

    My Goal Alignment: My goal for this website is to create a simple and functional design. I aim to make the buttons easy to find for users.The element of exploring will still be present however if the user wants to return to a certain page, they can do so with ease. Each page is setup to have a diiferent mood to it as if the user is travelling into different time zones.

    The interface elements that I used, make the user go through the whole page to find the back button. This has is pros and cons in this. The pro is that there is more engagement of the user end. The con is that if the user did not want that specific page they'll have to scroll through the whole page to go back, which can be frustrating.
